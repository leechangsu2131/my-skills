## 개요

나이스(NEIS)는 연가/출장 신청 화면을 팝업 모달로 띄우며, 내부에 강력한 보안 모듈과 커스텀 웹 컴포넌트를 사용하고 있어 일반적인 DOM 기반(HTML 요소) 제어 스크립트가 작동하지 않습니다.

따라서 다음과 같은 **하이브리드(반자동) 자동화 방식**을 사용합니다:
1. **사용자**: 직접 로그인
2. **셀레니움 (`neis_apply.py`)**: 나이스 메뉴를 클릭하여 '신청' 모달 창까지 자동으로 띄움
3. **GUI 매크로 (`neis_apply_gui.py`)**: 화면의 특정 좌표를 마우스로 클릭하고 키보드로 타입하여 신청 정보를 채움 (pyautogui 활용)

## 준비 사항

```text
파일 위치: skills/neis-leave-bot/
  ├── launch_chrome.bat        # 크롬 실행기 (원격 디버깅 포트 9222)
  ├── neis_apply.py            # [STEP 1] 메뉴 이동 및 신청창 열기 스크립트
  ├── neis_apply_gui_setup.py  # [STEP 2] 동적 캘리브레이션 툴 (원하는 좌표 무제한 찍기)
  ├── neis_apply_gui.py        # [STEP 3] 화면 좌표 기반 폼 자동 입력 스크립트 
  └── neis_test.py / neis_modal_map.py # 개발/디버깅용
```

### 필수 라이브러리 설치 (최초 1회)
```bash
pip install selenium webdriver-manager pyautogui
```

## 사용 방법

### 1단계: 특수 크롬 실행
`launch_chrome.bat` 을 더블클릭하여 실행합니다. 일반 크롬과 별도로 원격 제어용 나이스 창이 열립니다.
> ⚠️ 반드시 이 브라우저 창에서 **수동으로 인증서/보안 프로그램 로그인**을 완료해야 합니다.

### 2단계: 신청 창 띄우기 (Selenium)
로그인이 완료되면 터미널에서 다음을 실행하여 자동으로 신청 모달을 띄웁니다.
```bash
cd skills/neis-leave-bot
python neis_apply.py
```
👉 왼쪽 메뉴 `복무` ➡️ `개인근무상황관리` ➡️ 중앙의 `신청` 버튼을 자동으로 클릭합니다.

### 3단계: 캘리브레이션 (최초 1회만)
동적으로 폼의 순서를 구성하기 위해 아래 스크립트를 실행합니다. (권장: 크롬 창 최대화 상태)
```bash
python neis_apply_gui_setup.py
```
1. 스크립트가 실행되면 모달 빈공간, 휴가 드롭다운, 시작일, 종료일, 사유, 저장버튼 위치 등에 마우스를 올리고 `Enter`를 칩니다.
2. 각 위치를 찍을 때마다 해당 칸의 "이름(용도)"을 직접 타이핑해서 입력할 수 있습니다.
3. 모두 완료되면 `Ctrl+C`를 누릅니다.
4. 터미널에 출력되는 최종 파이썬 코드(`steps` 설정값 등)를 복사하여 `neis_apply_gui.py` 내의 `NeisGuiConfig.steps`에 붙여넣습니다.

### 4단계: 폼 자동 입력 (PyAutoGUI)
동적으로 구성한 `steps`에 따라 실제 휴가 신청을 자동화합니다. 
```bash
python neis_apply_gui.py
```
👉 설정해둔 좌표들을 순서대로 돌아다니며 미리 지정한 텍스트를 자동으로 입력하고 다음 칸으로 넘어갑니다.

## 스크롤 팁
- 스크롤을 내려서 버튼을 눌러야 하는 경우, `steps` 배열 리스트에 텍스트 값을 `"SCROLL_DOWN_3"`으로 주면 해당 좌표 클릭 후 자동으로 페이지다운 키를 3회 눌러줍니다.
- GUI 매크로가 실행되는 동안은 **마우스를 건드리지 마세요**.
- 위험 상황 대처: 매크로가 엉뚱한 곳을 클릭하면 **마우스 커서를 모니터 화면 제일 왼쪽 위(모서리)로 빠르게** 이동시키세요. (FailSafe 작동으로 즉시 중단됨)
