# 학생 기록 저장 워크플로우

선생님의 수업 메모, 클로바노트 텍스트, 또는 자유 메모를 학생별로 분류하고 노션 학생 기록 DB에 저장합니다.

## 사전 조건

- 노션 MCP 연동이 되어 있어야 합니다
- 학생 명단이 프로젝트 지침 또는 대화에 등록되어 있어야 합니다
- 노션에 학생 기록 DB가 이미 생성되어 있어야 합니다

---

## Step 1: 기록 텍스트 받기

사용자에게 정리할 텍스트를 요청합니다:

```
어떤 기록을 정리할까요? 아래 중 하나를 붙여넣어 주세요:
- 클로바노트 텍스트
- 수업 메모
- 자유 메모
- 구글폼 응답
```

날짜와 과목 정보도 함께 확인합니다.

---

## Step 2: 학생별 분류

`@student-record-classifier` 스킬의 지침에 따라 텍스트를 분류합니다:

1. 학생 명단 기반으로 텍스트에서 학생 이름 식별
2. 학생별로 내용 분리
3. 각 기록에 메타데이터 태깅 (날짜, 과목, 분야, 긍정도)
4. 구조화된 형태로 출력

---

## Step 3: 분류 결과 확인

분류 결과를 사용자에게 보여주고 확인을 받습니다:

```
위 분류 결과가 맞나요?
- 수정할 내용이 있으면 알려주세요
- "저장해줘" → 노션에 저장합니다
```

---

// turbo
## Step 4: 노션에 저장

확인이 완료되면 노션 학생 기록 DB에 저장합니다:

1. 노션에서 학생 기록 DB 찾기 (사전에 등록된 DB 사용)
2. 각 학생별 기록을 개별 행(row)으로 삽입
3. 속성 매핑: 학생명, 날짜, 과목, 분야, 내용, 긍정도, 출처
4. 저장 완료 보고

---

## Step 5: 완료 보고

사용자에게 결과를 보고합니다:

```
✅ 저장 완료!
- 총 ○명의 기록 ○건을 노션에 저장했습니다
- [학생 기록 DB 바로가기]
```

---

## 주의사항

- 한 번에 너무 많은 텍스트(A4 5장 이상)는 나눠서 처리하는 것이 좋습니다
- 학생 이름이 모호한 경우 사용자에게 확인 후 저장합니다
- 모둠 변경, 좌석배치 등 학급 전체 정보는 별도로 기록합니다
