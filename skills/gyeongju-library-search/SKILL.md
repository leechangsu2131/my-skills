# 경주시립도서관 재고 및 노션 연동 자동화 봇

이 스킬은 **노션(Notion)의 도서 신청 데이터베이스**의 책 목록(제목, 저자)을 읽어와서:
1. 교보문고 상품 검색을 통해 정확한 **ISBN-13**을 단일 요청으로 추출합니다.
2. 추출한 ISBN을 바탕으로 **경주시립도서관** 서버에 통합 검색을 수행합니다.
3. 검색된 수많은 결과들 중에서 소속된 도서관 지점('본관/시립', '송화', '단석', '전자' 등)을 취합하여, 다시 노션 데이터베이스의 다중 선택(Multi-select) 유형의 `도서관` 컬럼에 자동으로 꽂아 넣습니다.

## 필요 조건

- Python 3.8 이상
- `requests`, `beautifulsoup4` 라이브러리 설치
- 노션 API 통합(Integration) 토큰
- 타겟 도서신청 DB의 노션 페이지 ID

## 사용 방법

1. 라이브러리 설치:
   ```bash
   pip install -r requirements.txt
   ```
2. 환경 변수 세팅 (`.env` 파일 구성 또는 운영체제 환경변수로 설정):
   - `NOTION_TOKEN` = 'secret_...'
   - `NOTION_DATABASE_ID` = '313ddde1e1df80e9b161e2c5888f80d6'
3. 노션 DB 구조 확인 (스크립트 기본값):
   - 책 제목 텍스트: `제목` (Title 유형)
   - 책 저자 텍스트: `저자` (Rich Text 유형)
   - 업데이트 될 값: `도서관` (Multi-select 유형)
4. 스크립트 실행:
   ```bash
   python library_notion_sync.py
   ```

## 주의 사항
- `library_notion_sync.py` 스크립트 상단의 `NOTION_TOKEN` 변수 부분에 직접 토큰을 입력하셔도 됩니다.
- 검색 정확도를 극대화하기 위해 '제목'과 '저자' 컬럼의 데이터가 비교적 정확하게 입력되어 있으면 좋습니다. 저자가 없더라도 시도는 수행합니다.
