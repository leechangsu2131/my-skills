# 학기말 평가 문장 생성기 (Student Eval Generator)

## Overview

학기 동안 축적된 학생별 기록(수업 관찰, 생활 기록, 작품, 교우관계 등)을 종합 분석하여, 학기말 **교과 평가**와 **종합발달상황** 문장을 교육부 형식에 맞게 자동 생성합니다. 근거 없는 추상적 문구가 아닌, 실제 기록에 기반한 구체적 문장을 만듭니다.

## When to Use This Skill

- 학기말에 학생별 교과 평가 문장을 작성해야 할 때
- 종합발달상황 기록을 작성해야 할 때
- "○○ 학생 평가 써줘" 또는 "종합발달상황 문장 만들어줘"라고 요청할 때
- 특정 과목의 성취기준에 맞는 평가 문구가 필요할 때

## 사전 설정

### 필수 자료

1. **학생별 누적 기록**: 학기 동안 `student-record-classifier`로 축적된 기록
2. **성취기준** (선택): 3학년 교육과정 성취기준 PDF 또는 텍스트

### 평가 문장 스타일 가이드

모든 문장은 아래 원칙을 따릅니다:

| 원칙 | 설명 | 예시 |
|------|------|------|
| 3인칭 관찰자 | "~하였음", "~를 보임" | "수학적 사고력이 우수하며..." |
| 긍정 우선 | 부정 표현보다 성장 가능성 강조 | "~~노력이 필요함" → "~~에 관심을 갖고 꾸준히 노력하면 더욱 성장할 수 있음" |
| 구체적 근거 | 추상적 칭찬 대신 실제 상황 언급 | "잘함" → "도형 단원에서 삼각형과 사각형의 차이를 또래 친구에게 설명하는 등..." |
| 성장 변화 | 시간 경과에 따른 변화 서술 | "학기 초 ~~했으나, 후반기에는 ~~로 성장하였음" |
| 적절한 길이 | 교과별 2~4문장, 종합 5~8문장 | - |

## How It Works

### Step 1: 학생 기록 수집

해당 학생의 전체 기록을 수집합니다:

- **노션 DB 조회**: 학생 기록 DB에서 해당 학생 이름으로 필터링
- **대화 맥락**: 현재 프로젝트/대화에서 해당 학생 관련 기록 탐색
- **직접 입력**: 사용자가 추가 정보 붙여넣기

### Step 2: 기록 분석 및 정리

수집된 기록을 분석합니다:

```markdown
## 김민준 학생 기록 분석

### 📊 기록 개요
- 총 기록: 23건
- 기간: 2026-03-02 ~ 2026-07-15
- 긍정: 18건 (78%) | 중립: 3건 (13%) | 부정: 2건 (9%)

### 📚 과목별 정리
| 과목 | 기록수 | 주요 내용 |
|------|--------|-----------|
| 수학 | 8건 | 도형 이해 우수, 또래 교수 능력, 문제해결력 |
| 국어 | 5건 | 발표 적극적, 독서 습관 양호 |
| 과학 | 4건 | 실험 관찰 꼼꼼, 호기심 많음 |
| 기타 | 6건 | 모둠장 역할 수행, 교우관계 양호 |

### 📈 시간별 변화
- 3월: 수업태도 양호, 기초 확인 기간
- 4~5월: 수학 특기 두드러짐, 또래 교수 시작
- 6~7월: 리더십 성장, 모둠활동 주도
```

### Step 3: 평가 유형 선택

사용자에게 필요한 평가 유형을 물어봅니다:

1. **교과 평가**: 과목별 성취 평가 문장 (과목 선택)
2. **종합발달상황**: 전 영역을 아우르는 종합 기록
3. **특기사항**: 특정 영역의 특기/장점 기술
4. **전체**: 모든 유형 일괄 생성

### Step 4: 문장 생성

#### 교과 평가 예시 (수학)

```
수학적 사고력이 우수하며, 도형 단원에서 삼각형과 사각형의 성질 차이를 
정확히 이해하고 이를 또래 친구에게 설명하는 등 의사소통 능력이 돋보임. 
문제 해결 시 교과서에 제시된 방법 외에 자신만의 풀이 전략을 탐구하는 
탐구적 태도를 보이며, 모둠 활동에서 구성원들의 역할을 배분하고 
협력을 이끄는 리더십을 발휘함.
```

#### 종합발달상황 예시

```
밝고 적극적인 성격으로 학급 생활에 활발하게 참여하며, 특히 수학 
교과에서 뛰어난 이해력과 또래 교수 능력을 보임. 도형의 성질을 
스스로 탐구하고 자신만의 풀이 방법을 찾아 발표하는 등 수학적 
사고력과 창의성이 돋보임. 학기 초 모둠 활동에서 개인 활동을 
선호하는 모습을 보였으나, 학기 중반부터 모둠장 역할을 자발적으로 
맡아 구성원 간 역할 분배와 협력을 이끄는 리더십의 성장을 보임. 
독서에 대한 관심이 높아 학급 도서를 꾸준히 읽으며, 과학 실험 시 
관찰 기록을 꼼꼼하게 작성하는 등 학습 태도가 성실함. 친구들과의 
관계에서 배려심을 갖고 갈등 상황에서 중재 역할을 하려는 노력이 
인정됨.
```

### Step 5: 수정 및 확정

생성된 문장을 사용자에게 보여주고 수정 요청을 받습니다:

- "좀 더 짧게 줄여줘"
- "리더십 부분을 더 강조해줘"
- "부정적인 부분은 빼줘"
- "성취기준 ○○에 맞게 수정해줘"

수정 후 최종 확정된 문장을 노션 **평가 초안 DB**에 저장할 수 있습니다.

## Examples

### Example 1: 교과 평가 (국어)

**입력**: "이서연 학생 국어 교과 평가 써줘"

**해당 학생 기록 (자동 수집)**:
- 3/5: 국어시간 발표 소극적, 목소리 작음
- 4/10: 독서 감상문 내용 충실, 감정 표현 풍부
- 5/15: 모둠 토론에서 자기 의견 처음으로 적극 발표
- 6/20: 글쓰기 대회 참가, 생활문 작성 우수

**생성 결과**:
```
학기 초 발표 활동에 다소 소극적인 모습을 보였으나, 꾸준한 
독서 활동과 글쓰기 연습을 통해 자신감이 성장함. 독서 감상문에서 
등장인물의 감정을 자신의 경험과 연결 지어 표현하는 능력이 
우수하며, 학기 후반에는 모둠 토론에서 자신의 의견을 적극적으로 
발표하는 등 의사소통 능력의 뚜렷한 향상을 보임.
```

### Example 2: 전체 학생 일괄 생성

**입력**: "우리 반 전체 학생 종합발달상황 초안 만들어줘"

**결과**: 학생 번호순으로 개별 문장 생성 → 노션 평가 초안 DB에 일괄 저장

## Best Practices

- ✅ 학기 중 기록이 많을수록 더 구체적이고 근거 있는 문장 생성 가능
- ✅ 긍정적 기록뿐 아니라 성장 과정(부정→긍정 변화)도 기록하면 좋은 문장이 나옴
- ✅ 과목별 성취기준을 함께 제공하면 기준에 맞는 문장 생성 가능
- ✅ 생성된 문장은 반드시 선생님이 검토 후 수정
- ❌ AI가 생성한 문장을 그대로 사용하지 마세요 — 반드시 검토 필요
- ❌ 기록이 1~2건뿐인 학생은 문장이 빈약할 수 있음 — 추가 관찰 내용 입력 권장

## Common Pitfalls

- **Problem:** 기록이 부정적인 내용만 있는 학생
  **Solution:** 긍정적 재해석 + 성장 가능성 문구 활용. "~에 어려움을 보이나, ~에 관심을 가지면 성장 가능성이 높음"

- **Problem:** 비슷한 기록이 반복되어 문장이 단조로울 때
  **Solution:** 같은 강점이라도 시간별 발전 과정으로 서술. 다양한 표현 활용

- **Problem:** 성취기준과 실제 기록이 안 맞을 때
  **Solution:** 기록된 내용 중 성취기준과 연결 가능한 부분을 찾아 매핑
