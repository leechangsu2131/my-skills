# 평가 문장 출력 예시

## 교과 평가 예시

### 수학

```
수학적 사고력이 우수하며, 도형 단원에서 삼각형과 사각형의 성질 차이를 
정확히 이해하고 이를 또래 친구에게 설명하는 등 의사소통 능력이 돋보임. 
문제 해결 시 교과서에 제시된 방법 외에 자신만의 풀이 전략을 탐구하는 
탐구적 태도를 보이며, 모둠 활동에서 구성원들의 역할을 배분하고 
협력을 이끄는 리더십을 발휘함.
```

### 국어

```
학기 초 발표 활동에 다소 소극적인 모습을 보였으나, 꾸준한 독서 활동과 
글쓰기 연습을 통해 자신감이 성장함. 독서 감상문에서 등장인물의 감정을 
자신의 경험과 연결 지어 표현하는 능력이 우수하며, 학기 후반에는 모둠 
토론에서 자신의 의견을 적극적으로 발표하는 등 의사소통 능력의 뚜렷한 
향상을 보임.
```

## 종합발달상황 예시

```
밝고 적극적인 성격으로 학급 생활에 활발하게 참여하며, 특히 수학 
교과에서 뛰어난 이해력과 또래 교수 능력을 보임. 도형의 성질을 
스스로 탐구하고 자신만의 풀이 방법을 찾아 발표하는 등 수학적 
사고력과 창의성이 돋보임. 학기 초 모둠 활동에서 개인 활동을 
선호하는 모습을 보였으나, 학기 중반부터 모둠장 역할을 자발적으로 
맡아 구성원 간 역할 분배와 협력을 이끄는 리더십의 성장을 보임. 
독서에 대한 관심이 높아 학급 도서를 꾸준히 읽으며, 과학 실험 시 
관찰 기록을 꼼꼼하게 작성하는 등 학습 태도가 성실함.
```
