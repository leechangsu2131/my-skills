# 학생 사진 정리기 (Student Photo Organizer)

## Overview

교사가 촬영한 학생 작품 사진, 활동 사진을 학생별 폴더로 자동 분류합니다. 사진 파일이 있는 폴더를 지정하면, 학생 이름/번호 정보에 따라 하위 폴더를 생성하고 사진을 정리합니다. 정리된 결과를 노션에 기록할 수도 있습니다.

## When to Use This Skill

- 수업 후 찍은 학생 작품 사진을 학생별로 정리하고 싶을 때
- "이 사진들 학생별로 나눠줘" 또는 "작품 사진 정리해줘"라고 요청할 때
- 학생 포트폴리오용 사진을 폴더로 관리하고 싶을 때
- 사진과 함께 노션에 기록을 남기고 싶을 때

## How It Works

### Step 1: 폴더 및 사진 확인

사용자에게 다음 정보를 확인합니다:

1. **사진 폴더 경로**: 정리할 사진이 있는 폴더
2. **학생 매핑 방법**: 아래 중 하나
   - **파일명 기반**: 파일명에 학생 이름/번호 포함 (예: `01_김민준_미술.jpg`)
   - **순서 기반**: "1번째 사진은 1번 학생, 2번째는 2번 학생..."
   - **직접 지정**: 사용자가 하나씩 알려줌
   - **사진 내용 분석**: 사진에 적힌 이름을 읽기 (작품에 이름이 적혀있는 경우)
3. **과목/활동 정보**: "미술 수채화 작품" 등

### Step 2: 폴더 구조 생성

학생별 하위 폴더를 생성합니다:

```
학생작품/
├── 01_김민준/
│   ├── 2026-03-01_미술_수채화.jpg
│   └── 2026-03-15_과학_실험보고서.jpg
├── 02_이서연/
│   └── 2026-03-01_미술_수채화.jpg
├── 03_박지호/
│   └── 2026-03-01_미술_수채화.jpg
└── ...
```

**파일명 규칙**: `YYYY-MM-DD_과목_활동명.확장자`

### Step 3: 사진 복사/이동

사용자에게 복사(copy) 또는 이동(move)을 선택받습니다:
- **복사 (기본)**: 원본 유지, 학생 폴더에 복사본 생성
- **이동**: 원본을 학생 폴더로 이동 (원래 위치에서 삭제)

```bash
# 복사 예시
cp "/path/to/photo1.jpg" "/path/to/학생작품/01_김민준/2026-03-01_미술_수채화.jpg"
```

### Step 4: 결과 보고

```markdown
## 📸 사진 정리 완료 (2026-03-01 미술)

| 학생 | 파일 수 | 폴더 |
|------|---------|------|
| 01 김민준 | 1장 | 01_김민준/ |
| 02 이서연 | 1장 | 02_이서연/ |
| 03 박지호 | 1장 | 03_박지호/ |

📁 총 3명, 3장 정리 완료
```

### Step 5: 노션 기록 (선택)

"노션에도 기록해줘"라고 하면, 학생 기록 DB에 작품 사진 정보를 추가합니다:

```
학생: 김민준
날짜: 2026-03-01
과목: 미술
분야: 작품/활동
내용: 수채화 작품 완성 (사진 저장됨: 01_김민준/2026-03-01_미술_수채화.jpg)
긍정도: 긍정
```

## Examples

### Example 1: 파일명에 번호가 있는 경우

**사용자**: "~/Desktop/미술작품 폴더에 있는 사진들 학생별로 정리해줘"

```
~/Desktop/미술작품/
├── 1번_수채화.jpg
├── 2번_수채화.jpg  
├── 3번_수채화.jpg
└── 5번_수채화.jpg   ← 4번은 결석
```

**결과**:
```
학생작품/
├── 01_김민준/2026-03-01_미술_수채화.jpg
├── 02_이서연/2026-03-01_미술_수채화.jpg
├── 03_박지호/2026-03-01_미술_수채화.jpg
└── 05_정도윤/2026-03-01_미술_수채화.jpg

📊 4명 정리 완료, 4번 최수아 사진 없음 (결석?)
```

### Example 2: 순서대로 찍은 사진

**사용자**: "번호순으로 찍었어. 1번부터 8번까지 사진이야. 3번은 결석이라 없어."

이 경우 AI가 파일을 생성 시간순 정렬 → 1번, 2번, (3번 건너뜀), 4번... 순서로 매핑합니다.

## Best Practices

- ✅ 사진 촬영 시 번호순으로 찍으면 자동 매핑 가능
- ✅ 파일명에 학생 번호나 이름을 넣으면 가장 정확
- ✅ 작품에 이름을 쓰게 한 후 촬영하면 OCR로 식별 가능
- ❌ 여러 학생 작품을 한 장에 찍으면 분류 어려움 → 한 명씩 촬영
- ❌ 폴더에 사진 외 파일이 섞여 있으면 혼란 → 사진만 있는 폴더 사용

## Common Pitfalls

- **Problem:** 파일명에 한글이 깨지는 경우
  **Solution:** 파일명을 번호 기반(01, 02...)으로 변경 후 정리

- **Problem:** 사진 파일 형식이 HEIC(아이폰)인 경우
  **Solution:** HEIC 파일도 지원. 필요시 JPG 변환 옵션 제공

- **Problem:** 같은 학생의 사진이 여러 장인 경우
  **Solution:** 파일명에 순번 추가 (예: `2026-03-01_미술_수채화_01.jpg`, `_02.jpg`)
