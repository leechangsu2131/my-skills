# 학생 기록 자동 분류기 (Student Record Classifier)

## Overview

초등학교 교사가 클로바노트로 녹음한 텍스트, 자유 메모, 또는 구글폼 응답을 붙여넣으면, 텍스트 속 학생 이름을 식별하고 학생별로 내용을 분류합니다. 각 기록에 날짜, 과목, 분야, 긍정도를 자동 태깅하여 노션 DB에 바로 저장할 수 있는 구조화된 데이터로 변환합니다.

## When to Use This Skill

- 클로바노트로 녹음한 수업 관찰 내용을 학생별로 정리할 때
- 교실에서 빠르게 적은 메모를 학생 기록으로 분류할 때
- "오늘 수업 기록 정리해줘" 또는 "이 메모 학생별로 나눠줘"라고 요청할 때
- 구글폼 응답을 학생별로 정리할 때
- 노션의 학생 기록 DB에 일괄 저장하고 싶을 때

## 사전 설정

### 학생 명단 등록

프로젝트 지침 또는 대화 초반에 학생 명단을 등록합니다:

```
우리 반 학생 명단:
1번 김민준, 2번 이서연, 3번 박지호, 4번 최수아, 
5번 정도윤, 6번 한예은, 7번 오시우, 8번 강하린,
...
```

### 과목 목록 (3학년 기준)

```
국어, 수학, 사회, 과학, 도덕, 체육, 음악, 미술, 영어, 
창의적 체험활동, 안전한 생활
```

### 분류 분야

```
수업태도, 교과성취, 교우관계, 생활습관, 리더십, 
작품/활동, 특이사항, 모둠활동, 기타
```

## How It Works

### Step 1: 입력 텍스트 받기

사용자로부터 다음 형태 중 하나의 텍스트를 받습니다:

- **클로바노트 변환 텍스트**: 음성 녹음을 텍스트로 변환한 것
- **자유 메모**: 선생님이 직접 적은 메모
- **구글폼 응답 복붙**: 스프레드시트에서 복사한 텍스트

입력 시 날짜와 과목 정보가 없으면 사용자에게 물어봅니다:
- "이 기록의 날짜는 언제인가요?" (기본값: 오늘)
- "어떤 과목의 기록인가요?" (내용에서 추론 가능하면 자동 설정)

### Step 2: 학생 이름 식별

등록된 학생 명단을 기준으로 텍스트에서 학생 이름을 식별합니다:

- **정확한 이름 매칭**: "김민준" → 1번 김민준
- **이름만 매칭**: "민준이", "민준" → 1번 김민준
- **별명/약칭 매칭**: 프로젝트에 별명이 등록되어 있으면 활용
- **모호한 경우**: "준이" 같이 여러 학생에 해당할 수 있으면 사용자에게 확인

### Step 3: 학생별 내용 분리 및 태깅

각 학생에 대한 내용을 분리하고, 아래 속성을 자동 태깅합니다:

| 속성 | 설명 | 예시 |
|------|------|------|
| 학생 | 학생 이름 | 김민준 |
| 날짜 | 기록 날짜 | 2026-03-01 |
| 과목 | 관련 과목 | 수학 |
| 분야 | 기록 분류 | 수업태도 |
| 내용 | 구체적 기록 | 도형 개념을 친구에게 잘 설명함 |
| 긍정도 | 긍정/중립/부정 | 긍정 |
| 출처 | 자료 출처 | 클로바노트 |

### Step 4: 구조화된 결과 출력

결과를 아래 형식으로 출력합니다:

```markdown
## 📋 기록 분류 결과 (2026-03-01 수학)

### 1번 김민준
- **분야**: 수업태도 | **긍정도**: 😊 긍정
- **내용**: 도형 개념을 이해하고 옆자리 친구에게 자발적으로 설명해줌. 또래 교수 능력 우수.

### 2번 이서연
- **분야**: 수업태도 | **긍정도**: 😐 중립
- **내용**: 수업 초반 집중력이 다소 떨어졌으나 교사 개입 후 과제 수행함.

---
📊 총 2명 기록 | 긍정 1건, 중립 1건
```

### Step 5: 노션 저장 (선택)

사용자가 "노션에 저장해줘"라고 하면:

1. 노션의 **학생 기록 DB**를 찾습니다 (사전에 등록된 DB ID 사용)
2. 각 학생별 기록을 개별 행(row)으로 삽입합니다
3. 저장 완료 후 결과를 보고합니다

```
✅ 노션 저장 완료!
- 김민준: 1건 저장
- 이서연: 1건 저장
→ 학생 기록 DB 바로가기: [링크]
```

## Examples

### Example 1: 클로바노트 텍스트 분류

**입력**:
```
선생님: 오늘 3교시 수학시간인데요. 김민준 학생이 도형 단원에서 
삼각형과 사각형의 차이를 친구들한테 아주 잘 설명해줬어요. 
이서연 학생은 오늘 좀 피곤한 것 같았는데 집중력이 좀 떨어졌고요.
박지호는 모둠활동에서 모둠장 역할을 아주 잘 해줬습니다.
최수아는 수학 문제를 풀 때 자기만의 풀이 방법을 찾아서 발표했어요.
```

**출력**:
```markdown
## 📋 기록 분류 결과 (2026-03-01 수학)

### 1번 김민준
- **분야**: 수업태도 | **긍정도**: 😊 긍정
- **내용**: 도형 단원에서 삼각형과 사각형의 차이를 친구들에게 자발적으로 설명함. 또래 교수 능력 우수.

### 2번 이서연
- **분야**: 수업태도 | **긍정도**: 😟 부정
- **내용**: 피곤한 상태로 수업 집중력 저하 관찰됨.

### 3번 박지호
- **분야**: 모둠활동 | **긍정도**: 😊 긍정
- **내용**: 모둠장 역할을 적극적으로 수행함. 리더십 발휘.

### 4번 최수아
- **분야**: 교과성취 | **긍정도**: 😊 긍정
- **내용**: 수학 문제에서 독창적인 풀이 방법을 스스로 찾아 발표함. 수학적 사고력 우수.

---
📊 총 4명 기록 | 긍정 3건, 부정 1건
```

### Example 2: 자유 메모 분류

**입력**:
```
3월 1일 메모
- 민준이 서연이 싸움. 서연이가 먼저 사과함
- 지호 체육시간 축구에서 다리 살짝 다침 (보건실 다녀옴)
- 수아 미술 작품 완성도 높음, 게시판에 붙임
- 오늘 모둠 변경: 1모둠(민준,서연,시우), 2모둠(지호,수아,하린)
```

**출력**:
```markdown
## 📋 기록 분류 결과 (2026-03-01)

### 1번 김민준
- **분야**: 교우관계 | **긍정도**: 😟 부정
- **내용**: 이서연과 다툼 발생. (이서연이 먼저 화해함)

### 2번 이서연
- **분야**: 교우관계 | **긍정도**: 😐 중립
- **내용**: 김민준과 다툼 후 먼저 사과하여 화해함. 갈등 해결 노력 인정.

### 3번 박지호
- **분야**: 특이사항 | **긍정도**: 😟 부정
- **과목**: 체육
- **내용**: 축구 활동 중 다리 경미한 부상. 보건실 치료 후 복귀.

### 4번 최수아
- **분야**: 작품/활동 | **긍정도**: 😊 긍정
- **과목**: 미술
- **내용**: 미술 작품 완성도 우수. 학급 게시판에 게시됨.

### 📌 모둠 변경 기록
- 1모둠: 김민준, 이서연, 오시우
- 2모둠: 박지호, 최수아, 강하린

---
📊 총 4명 기록 | 긍정 1건, 중립 1건, 부정 2건
```

## Best Practices

- ✅ 학생 명단을 프로젝트 지침에 미리 등록해두면 매번 입력하지 않아도 됨
- ✅ 클로바노트 텍스트는 가능한 한 원문 그대로 붙여넣기 (AI가 정리함)
- ✅ 날짜와 과목 정보를 첫 줄에 간단히 적으면 분류가 더 정확함
- ✅ 모둠 변경, 좌석 변경 같은 학급 전체 정보도 함께 기록하면 맥락 파악에 도움
- ❌ 학생 이름 없이 "얘가 잘했음" 같은 표현은 분류 불가 → 이름을 포함해 주세요
- ❌ 한 번에 너무 많은 내용(A4 5장 이상)을 넣으면 누락될 수 있음 → 수업 단위로 나눠서 입력

## Common Pitfalls

- **Problem:** "준이"가 김민준인지 정도윤인지 모호할 때
  **Solution:** 모호한 이름은 사용자에게 확인 요청. 프로젝트에 별명 매핑 등록 가능

- **Problem:** 클로바노트가 이름을 잘못 인식했을 때 (예: "민준" → "민중")
  **Solution:** 유사 발음 이름 매핑을 프로젝트 지침에 추가. 자주 틀리는 패턴 기록

- **Problem:** 하나의 문장에 여러 학생이 언급될 때
  **Solution:** 각 학생별로 해당 내용을 적절히 분리하여 개별 기록으로 생성
