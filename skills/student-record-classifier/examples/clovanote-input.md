# 클로바노트 입력 예시

## 입력 1: 클로바노트 수업 관찰 (원문 그대로)

```
선생님: 오늘 3교시 수학시간이었는데요 도형 단원 삼각형 사각형 배우는 거였어요. 
김민준이가 삼각형이랑 사각형 차이를 옆에 앉은 이서연이한테 설명을 해줬는데 
아주 잘 설명을 해줬어요. 근데 이서연이는 오늘 좀 피곤했는지 수업시간에 
자꾸 딴짓을 했고요. 박지호는 모둠활동할 때 모둠장으로서 친구들한테 
역할 분배를 잘 해줬습니다. 그리고 최수아가 문제 풀 때 교과서에 없는 
자기만의 방법을 찾아가지고 발표도 했어요. 되게 창의적이었어요.
```

## 입력 2: 간단 메모

```
3/1 메모
- 1교시 국어: 민준이 발표 적극적, 도윤이 책 안가져옴
- 점심시간: 지호 서연 다툼 → 서연이가 먼저 사과
- 5교시 미술: 수아 작품 우수 (사진찍음), 시우 아직 미완성
```

## 입력 3: 구글폼 응답 정리

```
질문: 오늘 수업에서 재미있었던 점은?
김민준: 삼각형으로 집 만드는 거요
이서연: 색칠하는 거
박지호: 모둠에서 같이 하는 거
최수아: 새로운 방법 찾는 게 재미있었어요
```
